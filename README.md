Что это за репозиторий

АБИС — проект студенческой команды, который выполняется по заказу библиотеки учебного заведения. Здесь команда постепенно собирает автоматизированную систему учета книговыдачи: от исследования предметной области и требований до полноценного веб-приложения.

Важно: репозиторий устроен как история развития проекта по этапам. Документы лежат не «по этапам», а по типам артефактов. Для навигации по контрольным срезам есть отдельные индексные файлы со ссылками на актуальные документы и их версии.

О чем проект

Проект посвящен автоматизации учета книговыдачи в библиотеке учебного заведения.

Основные функции:

· 📚 Учет книжного фонда (каталог с обложками)
· 👥 Учет читателей (студенты, сотрудники)
· 📖 Выдача и возврат книг
· ⏰ Автоматический контроль просрочек
· 📊 Отчет «Просрочки» с фильтрацией
· 📥 Экспорт данных в CSV
· 🔐 Аутентификация и разграничение ролей (читатель, библиотекарь, администратор)
· 📱 Личный кабинет читателя

Что важно про интеграцию с 1С

По интеграции с 1С в данном проекте 1С рассматривается как внешний источник данных о книгах и читателях, а не как часть текущей версии решения.

На пилотном этапе проект развивается по шагам:

· сначала фиксируются данные, которые должны приходить извне (CSV-файлы);
· затем описываются форматы обмена и маппинг полей;
· только после этого появляется контрактный уровень интеграции.

В текущей версии проекта интеграция с 1С не реализована, но предусмотрена возможность импорта данных из CSV.

Правила ведения документации

· каждый важный документ начинается с короткой шапки: версия, статус, дата создания, дата обновления;
· после шапки идут краткие блоки «О документе», «Для кого» и «Основано на»;
· каждый документ получает стабильный код вида DOC-AREA-NNN;
· новые версии опираются на предыдущие, а не переписывают историю задним числом;
· индекс контрольной точки хранит список артефактов и ссылок на них, а не копии документов;
· решения из поздних этапов должны ссылаться на ранние артефакты;
· если команда меняет трактовку методики, формул или границ MVP, это фиксируется явно.
